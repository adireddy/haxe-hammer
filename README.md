haxe-hammer
===========

![haxe howler logo](https://raw.githubusercontent.com/adireddy/haxe-howler/master/logo.png)

Externs of hammer.js for Haxe - Javascript touch gestures library.

### Installation ###

```haxe
haxelib install hammerjs
```
